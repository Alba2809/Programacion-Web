3.2 Indique si cada uno de los siguiente enunciados es verdadero o false. Si es falso, explique por qué.

  1. Cualquier tipo input particular de formulario de HTML5 debe desplegarse de manera idéntica en todos los navegadores compatibles con HTML5.

    2. Falso, la forma que se despliegan los tipos input pueden variar de un navegador a otro.

  2. Cuando se coloca el enfoque en el campo de texto (es decir, si el cursor está en el campo de texto), se envía el texto del atributo placeholder al servidor.

    3. Falso, cuando se selecciona el campo de texto, el texto de placeholder desaperece y se envía el texto que escriba el usuario.

  3. No es necesario incluir el atributo autofocus en sus formularios.

    4. Verdadero.

  4. Los nuevos tipos input de HTML5 tienen validación automática del lado del cliente, con lo cual se elimina la necesidad de agregar secuencias de comandos (script) complicadas a sus formularios para validar la entrada del usuario y se reduce la cantidad de datos inválidos que se envín.

    5. Verdadero.

  5. El tipo input range se valida automáticamente de manera intrínseca cuando el navegador lo despliega como un control deslizante, ya que el usuario no puede mover el control fuera de los límites del valor mínimo o máximo.

    6. Verdadero.

  6. HTML5 cuenta con validación automática para el tipo input tel.

    7. Falso, los datos que se validan varín según la ubicación, por lo que sería bastante complejo realizar una validación automática.

  7. Si el usuario introduce un URL con un formato inapropiado en un tipo input url, no se validará. HTML5 no valida si el URL introducido existe en realidad.

    8. Verdadero.

  8. El elemento nav muestra un menú desplegable de hipervínculos.

    9. Falso, solo agrupa vínculos de navegación.

  9. El elemento header puede usarse sólo una vez en una página.

    10. Falso, se puede usar varias veces.

  10. Los elementos nav pueden anidarse en un elemento aside.

    11. Verdadero.

  11. Podemos usar el elemento brk para evitar dividir palabras en lugares incorrecto.

    12. Falso, se utiliza el elemento wbr.
