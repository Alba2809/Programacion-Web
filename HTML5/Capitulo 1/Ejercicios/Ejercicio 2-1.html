2.1 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. Una lista ordenada no puede anidarse dentro de una lista desordenada.

    2. Falso, todas las listas pueden estar anidadas para representar relaciones jerárquicas de los datos.

  2. El elemento br representa un salto de línea.

    3. Verdadero.

  3. Los hipervínculos se denotan mediante elementos link.

    4. Falso, el elemento para crear hipervínculos es a con el atributo href.

  4. La anchura de todas las celdas de datos en una tabla debe de ser igual.

    5. Falso, las celdas pueden tener diferente tamaño usando los atributos rowspan y colspan.

  5. Estamos limitados a un máximo de cinco vínculos internos por página.

    6. Falso, se pueden usar todos los hipervínculos que se deseen.
