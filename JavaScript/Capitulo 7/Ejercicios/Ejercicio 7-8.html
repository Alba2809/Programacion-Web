Â¿Qué está mal en la siguiente instrucción de repetición while?
while ( z >= 0 )
  suma += z;

Debido a que la variable z no es modificada en el cuerpo del while, se tendrá un ciclo infinito. Esto pasa sólo si la condición se cumple. Para evitarlo, z debe decrementarse para que pueda ser menor a 0.
