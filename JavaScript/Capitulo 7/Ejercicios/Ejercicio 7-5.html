Combine las instrucciones que escribió en el ejercicio 7.4 para formar una secuencia de comandos que calcule e imprima la suma de los enteros del 1 al 10. Use la instrucción while para iterar a través de las instrucciones de cálculo e incremento. El ciclo debe terminar cuando el valor de x se vuelva 11.

let suma = 0, x = 1;
while (x < 11) suma += x++;
document.writeln("<h3>La suma es: " + suma + "</h3>");
