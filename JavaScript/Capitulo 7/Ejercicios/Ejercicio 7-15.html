El proceso de encontrar el valor más grande (es decir, el máximo de un grupo de valores) se utiliza con frecuencia en aplicaciones de computadoras. Por ejemplo, una secuencia de datos para determinar el ganador de un concurso de ventas recibe como entrada el número de unidades vendidas por cada vendedor. El que haya vendido más unidades es el que gana el concurso. Escriba un algoritmo en pseudocódigo y después una secuencia de comandos que reciba como entrada una serie de 10 números de un solo dígito como caracteres, determine el mayor de los números y produzca un mensaje que muestre el número más grande. Su secuencia de comandos debe utilizar las siguientes tres variables:
a) contador: un contador para contar hasta 10 (es decir, para llevar el registro de cuántos números se han introducido, y para determinar que se hayan procesado los 10 números).
b) numero: el entero más reciente introducido por el usuario.
c) mayor: el número más grande encontrado hasta ahora.

Pseudocódigo:
Variables contador = 0, numero, mayor = 0;
Mientras contador sea menor a 10
  Recibir número de un digito
  Extraer primer caracter de la cadena  ⇐  Para tener solo un digito.
  Convertir numero en entero
  Si numero es > a mayor
    Reemplazar mayor igual a numero
  Incrementar contador
Imprimir el valor de mayor
Mostrar alerta con el valor de mayor

Código JavaScript:
let contador = 0, numero, mayor = 0;
while ( contador < 10 ){
  numero = window.prompt("Escriba un numero de un digito", "1");

  numero = numero.substr(0,1);

  numero = parseInt(numero);

  if(numero > mayor) mayor = numero;

  ++contador;
}
document.writeln("<h3>El número mayor es: " + mayor + "</h3>");
window.alert("El número mayor es: " + mayor);
