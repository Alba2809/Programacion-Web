Escriba una secuencia de comandos que obtenga del usuario el radio de un círculo y produzca texto de HTML5 que muestre el diámetro, la circunferencia y el área de ese círculo. Use el valor constante 3.14159 para π. [Nota: también puede usar la constante predefinida Math.PI para el valor de π. Esta constante es más precisa que el valor 3.14159. El objeto Math está definido por JavaScript y provee muchas herramientas matemáticas comunes]. Use las siguientes fórmulas (r es el radio): diámetro = 2r, circunferencia = 2πr, área = πr2.

var r = window.prompt("Escriba el radio: ", 0);

(r == "" || r == null) ? r = 0 : null;

r = parseFloat(r);

var diametro = 2 * r;

var circunferencia = 2 * Math.PI * r;

var area = Math.PI * (r * r);

document.write("Diámetro = " + diametro + "<br>");

document.write("Circunferencia = " + circunferencia + "<br>");

document.write("Área = "+ area + "<br>");
