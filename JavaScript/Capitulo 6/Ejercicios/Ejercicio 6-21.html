Escriba una secuencia de comandos que lea un entero, determine y produzca texto de HTML5 que muestre si es par o impar. [Sugerencia: use el operador residuo. Un número par es un múltiplo de 2. Cualquier múltiplo de 2 deja un residuo de cero al dividirse enter 2].

var n;

n = window.prompt("Número entero:", 0);

(n == "" || n == null) ? n = 0 : null;

n = parseInt(n);

((n % 2) == 0) ? document.write("<h3>" + n + " es un número par.</h3>") : document.write("<h3>" + n + " es un número impar.</h3>");
