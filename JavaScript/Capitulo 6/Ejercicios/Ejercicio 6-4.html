Identifique y corrija los errores en cada una de las siguientes instrucciones:

  1. if (c < 7);
        window.alert("c es menor que 7");

    2. El error es el ";", la forma correcta es:

    if (c < 7)
        window.alert("c es menor que 7");

    En la primera forma puede no haber error, pero si lo que se desea es que aparezca la alerta tomando en cuenta la condición, entonces se debe de quitar el ";".

  2. if (c => 7);
        window.alert("c es igual o mayor que 7");

    3. No ningún error.
