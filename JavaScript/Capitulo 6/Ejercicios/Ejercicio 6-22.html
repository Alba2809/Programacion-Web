Ejercicio 6.22

Alba García José Iván

Escriba una secuencia de comandos que lea dos enteros, determine y produzca texto de HTML5 que muestre si el primero es múltiplo del segundo. [Sugerencia: use el operador residuo].

var n1, n2;

n1 = window.prompt("Primer entero:", 1);
n2 = window.prompt("Segundo entero:", 1);

(n1 == "" || n1 == null) ? n1 = 1 : null;
(n2 == "" || n2 == null) ? n2 = 1 : null;

n1 = parseInt(n1);
n2 = parseInt(n2);

((n1 % n2) == 0) ? document.write(n1 + " es múltiplo de " + n2) : document.write(n1 + " no es múltiplo de " + n2);
