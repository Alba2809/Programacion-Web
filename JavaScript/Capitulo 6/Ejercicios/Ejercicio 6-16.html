Alba García José Iván

Escriba una secuencia de comandos que pida al usuario que esriba dos números, obtenga los dos números del usuario y produzca texto que muestre la suma, producto, diferencia y cociente.

var n1, n2;
n1 = window.prompt("Escriba el primer número:", 0);
n2 = window.prompt("Escriba el segundo número:", 0);
n1 = parseInt(n1);
n2 = parseInt(n2);
document.writeln("Suma = " + (n1 + n2) + "<br>");
document.writeln("Suma = " + (n1 * n2) + "<br>");
document.writeln("Suma = " + (n1 - n2) + "<br>");
document.writeln("Suma = " + (n1 / n2) + "<br>");
